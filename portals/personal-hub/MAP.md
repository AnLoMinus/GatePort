# 🗺 מפת פורטל אישי – "Gate of Self"

**עדכון אחרון:** 20.11.2025

---

## 🧭 מבנה הפורטל

```
פורטל אישי (GS)
├─ זהות אישית
│  ├─ מי אני
│  ├─ הערכים שלי
│  └─ החזון שלי
├─ ביוגרפיה
│  ├─ סיפורי חיים
│  ├─ נקודות ציון
│  └─ חוויות משמעותיות
├─ יעדים
│  ├─ מטרות קצרות טווח
│  ├─ מטרות בינוניות טווח
│  └─ מטרות ארוכות טווח
├─ יומן פרויקטים
│  ├─ פרויקטים פעילים
│  ├─ פרויקטים הושלמו
│  └─ פרויקטים עתידיים
└─ קישורים אישיים
   ├─ פרופילים ברשתות
   └─ אתרים אישיים
```

---

## 📍 תתי־שערים

### 1. זהות אישית
**מיקום:** `identity/`

**תיאור:**  
מקום לזהות האישית, הערכים, והחזון. מי אני, מה חשוב לי, ולאן אני הולך.

**תוכן:**
- הצהרת זהות
- ערכים מרכזיים
- חזון אישי
- פילוסופיה אישית

---

### 2. ביוגרפיה
**מיקום:** `biography/`

**תיאור:**  
סיפורי החיים, נקודות הציון, והחוויות המשמעותיות שעיצבו את הדרך.

**תוכן:**
- סיפורי חיים
- נקודות ציון חשובות
- חוויות משמעותיות
- לקחים נלמדים

---

### 3. יעדים
**מיקום:** `goals/`

**תיאור:**  
המטרות האישיות, התוכניות, והחלומות לעתיד.

**תוכן:**
- מטרות קצרות טווח (3 חודשים)
- מטרות בינוניות טווח (6-12 חודשים)
- מטרות ארוכות טווח (12+ חודשים)
- חלומות וחזונות

---

### 4. יומן פרויקטים
**מיקום:** `projects-journal/`

**תיאור:**  
מעקב אחרי הפרויקטים האישיים: פעילים, הושלמו, ועתידיים.

**תוכן:**
- פרויקטים פעילים
- פרויקטים שהושלמו
- פרויקטים עתידיים
- לקחים וסיכומים

---

### 5. קישורים אישיים
**מיקום:** `links/`

**תיאור:**  
קישורים לפרופילים ברשתות, אתרים אישיים, ופלטפורמות שונות.

**תוכן:**
- פרופילים ברשתות חברתיות
- אתרים אישיים
- בלוגים
- פלטפורמות אחרות

---

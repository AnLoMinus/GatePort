# 🌀 פורטל אישי – "Gate of Self" (GS)

**קוד:** `GS`  
**גרסה:** v0.0.1  
**סטטוס:** סקיצה

---

## 📖 תיאור

הפורטל האישי הוא השער לזהות, לביוגרפיה, לסיפורי החיים, ליעדים ולחזון האישי. כאן מתרכזים כל הפרויקטים האישיים, היומנים, והמסמכים האוטוביוגרפיים.

---

## 🎯 מטרה

ליצור מקום אחד שמרכז את כל ההיבטים האישיים: זהות, ביוגרפיה, חזון, יעדים, ופרויקטים אישיים.

## 📐 מבנה פורטל סטנדרטי

כל פורטל מכיל 3 קבצים מרכזיים:

### 1. `README.md`
- תיאור הפורטל
- מטרות ותחומי אחריות
- חלוקה פנימית
- הוראות שימוש

### 2. `LINKS.md`
- קישורים למאגרים קשורים
- קישורים לריפואים אחרים
- קישורים חיצוניים רלוונטיים
- ארגון לפי קטגוריות

### 3. `MAP.md`
- מפת תתי־שערים בתוך הפורטל
- היררכיה של תוכן
- קשרים בין חלקים שונים
- ניווט פנימי

## 🔄 זרימת מידע

```
משתמש
  ↓
README.md (פורטל ראשי)
  ↓
PORTALS_INDEX.md
  ↓
פורטל ספציפי (portals/[name]/)
  ↓
README.md (פורטל)
  ↓
LINKS.md / MAP.md
  ↓
מאגרים חיצוניים / תתי־שערים
```

## 🔗 קשרים בין פורטלים

### קשרים ישירים
- פורטל מפתחים ↔ פורטל עיצוב (פרויקטים משותפים)
- פורטל משחקים ↔ פורטל SparKing (קלפים ומשחקים)
- פורטל מאגרי ידע ↔ פורטל פסיכו־רוחני (תוכן רוחני)

### קשרים עקיפים
- כל הפורטלים מקושרים דרך PORTALS_INDEX.md
- חיפוש מאוחד (עתידי)
- API פנימי (עתידי)

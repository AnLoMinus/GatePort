# 📁 תיקיית Assets

## 🖼️ תמונת רקע

לאחר יצירת תמונת הרקע עם ChatGPT/DALL-E/Midjourney:

1. שמור את התמונה בשם `background.jpg` (או `.png` / `.webp`)
2. העלה אותה לתיקייה הזו (`website/assets/`)
3. האתר יזהה אותה אוטומטית וישתמש בה כרקע

## 📝 פורמטים מומלצים

- **JPG** - טוב לרקע עם צבעים רבים (קטן יותר)
- **PNG** - טוב אם יש שקיפות
- **WebP** - הכי מומלץ (קטן ביותר, איכות מעולה)

## 🎨 מידות מומלצות

- **מינימום:** 1920x1080 (Full HD)
- **מומלץ:** 3840x2160 (4K)
- **אופטימלי:** 5120x2880 (5K)

## 📦 אופטימיזציה

לפני העלאה, ודא שהתמונה:

- אופטימזית (קבצים קטנים יותר = טעינה מהירה יותר)
- בגודל מתאים (לא יותר מדי גדול)
- בפורמט WebP אם אפשר

## 📋 דרכי תרומה

### 1. דיווח על באגים
- פתח Issue עם תיאור מפורט
- צרף צילומי מסך אם רלוונטי
- ציין גרסה וסביבה

### 2. הצעות שיפור
- פתח Issue עם תיאור הרעיון
- הסבר למה זה יעזור
- הצע דרך יישום

### 3. תיקונים ותוספות
- Fork את המאגר
- צור branch חדש
- בצע את השינויים
- פתח Pull Request

### 4. תיעוד
- שיפור תיעוד קיים
- הוספת דוגמאות
- תרגום (אם רלוונטי)

### 5. בדיקות
- בדיקת קישורים
- דיווח על קישורים שבורים
- הצעות לשיפורים

## 🔀 תהליך Pull Request

### 1. הכנה
- Fork את המאגר
- Clone ל־local
- צור branch חדש: `git checkout -b feature/your-feature-name`

### 2. פיתוח
- בצע את השינויים
- ודא שהכל עובד
- עדכן תיעוד אם נדרש

### 3. בדיקה
- בדוק שהקוד נקי
- ודא שאין שגיאות
- בדוק קישורים אם הוספת

### 4. שליחה
- Commit עם הודעה ברורה
- Push ל־branch שלך
- פתח Pull Request

### 5. סקירה
- המתין לסקירה
- הגב על הערות
- בצע שינויים אם נדרש
